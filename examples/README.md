# SLAM demo
Demonstration of the SLAM system.
# Usage
`<EXECUTABLE> [options] <DATASET_PATH>`
# Optional parameters
```bash
  --help                       Produce help message
  --data-format arg (=icl_tum) Format of dataset (tum, icl, icl_tum)
  --keyframes-step arg (=10)   Step between keyframes
```
